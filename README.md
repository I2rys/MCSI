<h1 align="center">MCSI</h1>
<h4 align="center">Minecraft server information grabber</h4>

## Installation
Github:
```
git clone https://github.com/I2rys/MCSI
```

NPM Packages:
```
npm i request-async
```

## Usage
```
node index.js <serverIP>
```

 - serverIP - The target server(IP).
